# Apache Flink (apache-flink)
Apache Flink is a framework and distributed processing engine for stateful computations over unbounded and bounded data streams.

## APIs

### Apache Flink REST API
The REST API provides programmatic access to monitor and control Flink jobs and clusters. It supports job submission, cluster management, and metrics retrieval.

**Human URL:** [https://flink.apache.org/](https://flink.apache.org/)
