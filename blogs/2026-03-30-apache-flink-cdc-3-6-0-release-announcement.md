The Apache Flink Community is excited to announce the release of Flink CDC 3.6.0! This release extends Flink version support to 1.20.x and 2.2.x, upgrades JDK version to 11, introduces new Oracle Source and Apache Hudi Sink Pipeline connectors, adds Lenient mode schema evolution support for Fluss Pipeline connector, introduces PostgreSQL Schema Evolution support, enhances Schema Evolution capabilities for better multi-table synchronization scenarios and table name mapping flexibility, and strengthens Transform framework with VARIANT type and JSON parsing support.
